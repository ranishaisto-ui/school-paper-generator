A professional exam paper generator plugin for schools and educational institutions.

## Installation

1. Upload the `school-paper-generator` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Paper Generator' in admin menu to start

## Usage

### 1. Add Questions
- Go to 'Question Bank' in admin
- Click 'Add New Question'
- Select question type (MCQ, Short, Long)
- Enter question text and details
- Save question

### 2. Create Paper
- Go to 'Create Paper' in admin
- Select subject and class level
- Choose questions from question bank
- Add school details
- Generate paper

### 3. Export Paper
- Click 'Export' button
- Choose format (PDF, Word, Excel)
- Download and print

## Shortcodes

`[spg_paper id="123"]` - Display paper with ID 123  
`[spg_question_bank]` - Display question bank for students

## Requirements

- WordPress 5.0 or higher
- PHP 7.2 or higher
- MySQL 5.6 or higher
